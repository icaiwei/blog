    # 库函数和寄存器

    # 区别

    ​ 本质上是一样的。我们可以在库函数模板里面，直接操作寄存器，因为官方库相关头文件有寄存器定义。但是不能在寄存器模板调用库函数，因为没有引入库函数相关定义。 了解寄存器基本原理的目的是为了让我们对STM32相关知识有比较深入的理解，这样在开发过程中方可得心应手，游刃有余。底层代码配置出了问题需要调试的话，必须对寄存器有一定的了解才能找到问题，因为调试代码，底层只能查看寄存器相关配置。

    # 工程结构

    # 文件夹

    Common、Main、Project、Startup_config、STM32F4_FWLIB、USER、XXXX 由以上的文件夹组成一个程序工程结构 注：“XXXX”这个文件夹根据用户应用程序命名

    # 介绍

    1. “Common”:存放工程共用文件,延时函数、位操作

    2. “Main”：存放工程的主函数main.c

    3. “Project”：存放工程的打开文件、编译产生的中间文件和hex 文件

    4. “Startup_config”:存放STM32 的启动文件、核心文件和一些头文件

    5. “USER”:存放STM32 外设的驱动文件，如：led、key、lcd、ADC、DAC、PWM ……

    6. “STM32F4_FWLIB”：存放STM32 库文件，如使用STM32F1 芯片，文件夹命名为：STM32F1_FWLIB；使用STM32F7 芯片，文件夹命名：STM32F7_FWLIB

    7. “XXXX”：存放用户运用程序文件(该文件夹可多个，依据用户应用程序多少决定)，如使用文件系统，就命名：FATFS；使用网络运用，就命名：LWIP ……

    # 固件库模板

    # 初始

    环境：MDK5

    固件库版本：v1.4(F4专用)

    重要的预编译全局宏定义标识符：

    STM32F40_41xxx,USE_STDPERIPH_DRIVER

    启动文件：

    startup_stm32f40_41xxx.s

    # 常见错误

    "no source": Error: command-line: #992: invalid macro definition: STM32F40_41xxx.USE_STDPERIPH_DRIVER

    答案：STM32F40_41xxx,USE_STDPERIPH_DRIVER 
    宏定义标识符错误。。。。输入要正确，逗号隔开

    ..\FWLIB\src\stm32f4xx_usart.c(1038): warning: #223-D: function "assert_param" declared implicitly

    答案：宏定义标识符“USE_STDPERIPH_DRIVER”书写错误。

    ..\USER\stm32f4xx.h(102): error: #35: #error directive: "Please select first the target STM32F4xx device used in your application (in stm32f4xx.h file)“

    答案：宏定义标识符“STM32F40_41xxx”书写错误。

    ..\USER\stm32f4xx_conf.h(34): error: #5: cannot open source input file "stm32f4xx_adc.h": No such file or directory

    头文件路径找不到。加入path的路径为inc子目录，不是src子目录。
或者压根头文件路径inc就没有加入到工程的path中。

    ; Reset handler
Reset_Handler    PROC
                 EXPORT  Reset_Handler             [WEAK]
;IMPORT  SystemInit	
;寄存器代码,不需要在这里调用SystemInit函数,故屏蔽掉,库函数版本代码,可以留下
;不过需要在外部实现SystemInit函数,否则会报错.
IMPORT  __main
    LDR     R0, =0xE000ED88    ; 使能浮点运算 CP10,CP11
    LDR     R1,[R0]
    ORR     R1,R1,#(0xF << 20)
    STR     R1,[R0]

    ;LDR     R0, =SystemInit	;寄存器代码,未用到,屏蔽
    ;BLX     R0				;寄存器代码,未用到,屏蔽
    LDR     R0, =__main
    BX      R0
    ENDP
